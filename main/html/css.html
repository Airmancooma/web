DKT Programozási alapok: Webszerkesztési alapok stíluslapok használata

A stíluslapok használata azt jelenti, hogy egy HTML oldalhoz egyszerűen hozzáadhatunk egy CSS stíluslapot, amely megadja az oldal megjelenésének a beállításait. Ez a szabványos megjelenésű oldalak fejlesztésének kulcsa, mivel lehetővé teszi a kód újrakeresését és szétszórását. A stíluslapok használata HTML és CSS parancsokkal történik.

CSS tagek

A CSS parancsok segítségével lehetővé válik az oldal megjelenésének szabályozása. A leggyakrabban használt CSS parancsok a következők:

  • font-family - betűtípusok megadása
  • Serif - elsősorban nyomtatásban használt, talpas betűtípus
  • Sans-serif - a képernyőn jól mutató, modern megjelenésű, talp nélküli betűtípusok
  • Monospace - azonos szélességű karaktereket használó, írógéppel írt hatást keltő betűtípus család
  • Cursive - kézírásra emlékeztető, nehezen olvasható, elsősorban dekoratív célokat szolgáló betűtípus
  • font-weight
  • font-size - betűméret megadása
  • Abszolút méret megadása:
    • Kulcsszavak használata - növekvő sorrendben: xx-small, x-small, small, medium(alapértelmezett), large, x-large, xx-large, xxx-large
    • hosszúság-mértégegységek használata (px)

  • Relatív méret megadása:
    • Kulcsszavak használata - smaller(kisebb), larger (nagyobb)
    • Relatív mértékegységek használata (em, %)
  • color - szöveg színének megadása
  • background-color - háttérszín megadása
  • margin - margók megadása
  • padding - párnázás megadása
  • border - szegély megadása
  • in - inch(hüvelyk)
  • cm - centiméter
  • pt - pont
  • px - pixel(képpont)
  • em - az aktuélis betűmérethez képest megadott méretet jelzi (1.2em az aktuális betűméretet 1,2-szeresére növeli
  • % - széleskörben használható mértékegység
  • rem - a gyökérelem (a HTML-oldalánál a html elem) betűméretéhez képest megadott méretét jelzi
  • color - szín beállítása
  • # - hexadecimális színkód
  • rgb - színkordináta-rendszer
  • hsl - színkordináta-rendszer
  • oblique - dőlt stílust eredményez az eredeti betűk számítógép általi megdöntésével
  • italic - az aktuális betűtípus dőlt variánsát jeleníti meg, ha van ilyen a betűkészletben
  • normal - a betűk stílusát normál stílusra állítja (vissza)
  • strong - erős kiemelés
  • font-variant betűváltozatok beállítása pl.: small-caps
  • font-stretch: - betűk nyújtása

    • Sűrítés:
      • ultra-condensed - 50% - 50% sűríti a betűket
      • extra-condensed - 62.5% - 62.5% sűríti a betűket
      • condensed - 75% - 75% sűríti a betűket
      • semi-condensed - 87.5% - 87.5% sűríti a betűket

    • Normál:
      • normal - 100% - normál

    • Ritkítás:
      • semi-condensed - 112.5% - 112.5% ritkítja a betűket
      • expandded - 125% - 125% ritkítja a betűket
      • extra-expanded 150% - 150% ritkítja a betűket
      • ultra-expanded - 200% - 200% ritkítja a betűket
  • Szegélyek:
    • solid - folytonos szegély
    • dotted - pontozott szegély
    • dashed - szaggatott szegély
    • double - dupla szegély
    • grovove - barázdált hatású szegély
    • ridge - a kjépernyő síkjához képest kiemelt hatású szegély
    • inset - süllyesztett hatású (3D) dobozt eredményez
    • outset - kiemelt hatású (3D) dobozt eredményez
  • text-align - szöveg igazítása (left, center, right, justify)
  • text-decoration - szöveg díszítése (underline, overline, line-through, none)
  • line-height - sor magasságának beállítása
  • word-wrap - szavak átcsoportosítása a következő sorra, ha nem férnek el a jelenlegi sorban
  • white-space - szöveg alapértelmezett sortörésének beállítása (normal, nowrap, pre)
  • display - elem megjelenésének beállítása (block, inline, inline-block, flex, grid, none)
  • float - elem flottálása (left, right, none)
  • clear - elem flottálási tulajdonságainak törlése (left, right, both, none)
  • position - elem pozícionálásának beállítása (static, relative, absolute, fixed)
  • top, bottom, left, right - elem pozícionálásának beállítása (pl. position: absolute; top: 20px; left: 30px;)
  • overflow - elem tartalmának kibuggyanásának kezelése (visible, hidden, scroll, auto)
  • z-index - elemek egymás feletti rendezésének beállítása
  • box-sizing - elem méretének beállítása (content-box, border-box)
  • Flex:
    • flex-direction - flex elemek irányának beállítása (row, row-reverse, column, column-reverse)
    • justify-content - flex elemek igazítása a tengely mentén (flex-start, flex-end, center, space-between, space-around)
    • align-items - flex elemek igazítása a kereszt tengely mentén (flex-start, flex-end, center, baseline, stretch)
    • align-content - több soros flex elemek igazítása a kereszt tengely mentén (flex-start, flex-end, center, space-between, space-around, stretch)
    • flex-wrap - flex elemek új sorba rendezése, ha nem férnek el a jelenlegi sorban (nowrap, wrap, wrap-reverse)
    • order - flex elemek megjelenési sorrendjének beállítása
  • List-style
    • none - nem jelenik meg semmi
    • disc - egy kis korong jelenik meg
    • circle - kört rajzol a listaelemekhez
    • square - négyzetet rajzol a listaelemekhez
    • decimal - számokkal jelöli a listaelemeket
    • decimal-leading-zero - számokkal jelöli a listaelemeket, és nullákkal tölti ki a rövidebb számokat
    • lower-alpha - kisbetűs betűsorozat jelöli a listaelemeket
    • upper-alpha - nagybetűs betűsorozat jelöli a listaelemeket
    • lower-roman - római számokkal jelöli a listaelemeket
    • upper-roman - nagybetűs római számokkal jelöli a listaelemeket
    • lower-greek - görög betűkkel jelöli a listaelemeket
    • upper-greek - nagybetűs görög betűkkel jelöli a listaelemeket
    • lower-latin - latin betűkkel jelöli a listaelemeket
    • upper-latin - nagybetűs latin betűkkel jelöli a listaelemeket

Ezek a parancsok lehetővé teszik, hogy az oldal megjelenését megváltoztassuk, hogy jobban illeszkedjen a célközönség igényeihez.
